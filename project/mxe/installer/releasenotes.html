Thank you for installing digiKam!

We would like to inform you that this port of digiKam under Windows is not as stable as the Linux version.

digiKam works mostly fine under Windows, but some features are still broken due to some bugs or not yet implemented features in the underlying KDE.org libraries.

To handle video files with digiKam under Windows, we need an extra and free video codec package, as KLite Codec Pack for exemple.

To report a bug or a whish, please follow instructions at digiKam project web site.

To know all current reports open about Windows release, please check the relevant the Bugzilla entries.

NOTE: when a dysfunction is discovered please get a backtrace of debug statements generated by digiKam while running.
Not like under Linux, these information are printed on a system console only visible with a dedicated application named DebugView.
You can donwload this tool at this url.

Thanks in advance for your support.

The digiKam team
